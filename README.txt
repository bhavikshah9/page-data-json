
-- SUMMARY --

Page Data JSON module allows you to represent Page data in JSON format.


-- REQUIREMENTS --

None.


-- INSTALLATION --

* Install as usual.

  See https://www.drupal.org/docs/7/extending-drupal/installing-modules for
  further information.


-- STEPS --

1 Activate module.
2 Add Site API key. Visit /admin/config/system/site-information.
3 Start accessing Page data in JSON format.
    URL format: http://localhost/page_json/<API-key-here>/<node-ID-here>
